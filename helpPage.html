Create an Equation

To create an image out of your equation, first select the math equation format of your choice (Ex latex,MathML,Ascii). Then type your equation inside the text box. As you're typing youâ€™ll see a preview image of what your image will look like.. Once you are done hit the submit button and it will create an image of the equation. Just a note as you are typing the preview will always be very small and black no matter what your settings. This is to increase usability with a small ui.

Copy Button

will move the create equation onto the slideshow

bind Button

This will bind the equation to the UI. So when you hit the copy button it will replace the old image with the new image

unbind button

This button will unbind the equation to the UI. So when you hit the copy button it will add a new image

No data is collected

This app doesn't collect any data. The permission to use this app are only used for moving the image onto the google slide. This project is 100% open source and was created to help people out. You can view the code here
